# Instacart Customer Behavior Analysis (Python)

This project analyzes customer purchasing behavior using Instacart order data as part of the CareerFoundry Data Analytics Immersion program.

## Business Objective
Instacart wants to better understand how customers shop, reorder products, and interact with departments in order to improve marketing strategy and product recommendations.

## Key Questions Answered
- Which departments and products are most frequently purchased?
- How often do customers reorder the same products?
- What ordering patterns exist by time of day and day of week?
- Which product categories show the strongest customer loyalty?

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebooks
